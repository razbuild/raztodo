## [0.1.0] - 2025-01-01

### Added
- Initial release
- Task management: add, update, remove, list, done
- Search tasks by keyword
- Priority levels (L/M/H)
- Due dates support
- Tags and projects
- JSON import/export
- Database migration command
- Colored terminal output (ANSI)
- Cross-platform support (Linux, macOS, Windows)
- SQLite storage (zero dependencies)
